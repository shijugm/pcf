This project will cover
    using python with a database
    importing data into the database - csv, json, xml
    exporting data from the database - csv, json, xml

Database
    This code uses sqllite which is available by default in mac. Change the connection details in the
    conn module to use other dbs

SQLLITE setup
    create new database  : sqlite3 pcf.db
    connect to an existing sqlitedb from the terminal using - sqlite3 pcf.db
    place the db file in the data folder. Or update the database path in the connection module
